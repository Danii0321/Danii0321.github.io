Bubble Run Universe

A more advanced version of Bubble Run, now in 3D! Also created in Processing. Rather than a flat plane, the game is mapped to the surface of a cube that can be rotated around the player's avatar. The objective is still to collect all the coins (8 per level) and avoid enemies, but there are now levels called worlds. Each world increases in difficulty (more enemies) and can only be played once previous worlds are beat. I also used the color and shading features to make the game more visually interesting.

Bubble Run

Bubble Run was created in Processing. It is built by PShapes for the coins and enemies, and has several game states corresponding to playing, winning, and losing. The user plays as a small white ball, and controls its movement with their cursor. The goal of the game is to collect all 10 coins while avoiding the red enemies. The enemies move based on specific velocity values and "bounce" on the border instead of going out of the frame. The start screen contains instructions for playing the game. Try it out!
